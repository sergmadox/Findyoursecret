Как то на работе дали задание просмотреть файлы и папки на содержание в них:

1) Книг
2) Медиа файлов
3) Музыки
4) Исполняемых файлов и скриптов
5) Текстовых документов, содержащие пароли или логины
6) Программного обеспечения

Для этого и был написан данный скрипт. Он состоит из нескольких функций:

extention($Path) - Функция поиска по расширениям исполняемых файлов и скриптов;

bookin($Path) - Функция поиска по расширениям медиа-файлов;

filtered($Path) - /-/-/-/ по расширениям текстовых файлов и поиска по средствам регулярного выражения слов: Логин, пароль и т.д.

Запуск скрипта.
Вариант 1.
	Положить скрипт в нужный каталог и запустить

Вариант 2. 
	Через ключ -Path. Пример PS D:\>.\Findyoursecret.ps1 -Path D:\

После запуска скрипта формируются три *.CVS файла, содержащий полный путь до найденного файла.
